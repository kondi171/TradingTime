## Features

- Themes change
- Wallpaper change
- Avatar change
- Calculating the probability of buying / selling actions
- Various types of modals 
- Fetching REST API
- Extensive database 


## Tech Stack

**Base:** HTML, JS, CSS

**Frameworks:** React, React Router, Chart

**Preprocessor:** SASS/SCSS

**Server:** PHP, MySQL
